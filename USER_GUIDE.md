### Menambah Tamu Baru

1. Klik tombol **+ Tambah**
2. Isi form:
   - Guest ID (wajib)
   - Nama (wajib)
   - Lokasi Meja
   - Perusahaan
   - Divisi
   - Foto (opsional)
3. Klik **Simpan**

### Import Tamu dari Excel/CSV

1. Siapkan file CSV dengan format:
```csv
guestId,name,tableLocation,company,division
G001,John Doe,Table A1,PT ABC,Marketing
G002,Jane Smith,Table A2,PT XYZ,Finance
```

2. Klik tombol **Import CSV**
3. Pilih file CSV
4. Klik **Upload**

### Export Data Tamu

1. Klik tombol **Export**
2. Pilih format (CSV)
3. File akan terdownload
